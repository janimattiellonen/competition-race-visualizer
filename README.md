# Competition Race Visualizer

A retro-styled disc golf competition visualizer that presents tournament results as an animated "horse race". Players move left-to-right across a neon track, with speed determined by their scores relative to par.

## Features

- Fetches competition data from Disc Golf Metrix
- Synthwave/outrun aesthetic with neon glows and retro fonts
- Animated race visualization with per-hole score popups
- Results podium with full leaderboard

## Tech Stack

- React 19 + TypeScript
- Vite 8
- Framer Motion 12

## Getting Started

```bash
nvm use
npm install
npm run dev
```

Dev server runs on http://localhost:5131
